// WHAT IS GO?

Go is a cross-platform, open source programming language.
It is used to create high-performance applications
Go compared to other programming languages like Python and C++ is a fast statically typed compiled langauge.
Go has syntax similar to C++.
Lastly, it was developed at Google by Robert Griesemer, Rob Pike and Ken Thompson in the year 2007.

// WHAT IS GO USED FOR EXACTLY

Web Development
Developing cross-platform applications(apps that can run on multiple OS)
Developing network based programs
Also Cloud-native development

// COMPARISON OF GO TO PYTHON & C++

Feature,         Go (Golang),             Python,                    C++
Type System,     Statically typed,        Dynamically typed,         Statically typed
Execution,       Compiled,                Interpreted,               Compiled
Run Time,        Fast,                    Slow,                      Fast
Compile Time,    Fast,                    N/A (Interpreted),         Slow
Concurrency,     Goroutines & Channels,   No built-in mechanism,     Supports threads
Memory,          Garbage Collection,      Garbage Collection,        Manual Management
OOP Classes,     No (Uses Structs),       Has classes & objects,     Has classes & objects
Inheritance,     Does not support,        Supports inheritance,      Supports inheritance
